Basics Of CSS

CSS Stands for cascading style sheet used to style web pages.

Syntax

Selector
Used to select an element or a set of elements
Property
Used to specify the property of the element
Value
Used to specify the value of the property
        
            h1{
                color: red;
            }
            here h1 is selector, color is property and red is value

Selectors

Universal Selector like *
Used to select all elements
Group Selector i.e. html,body
Used to select multiple elements
Class Selector
Used to select elements with a specific class
Id Selector
Used to select elements with a specific id

Colors

RGB
Used to specify a color in terms of red, green, and blue
RGBA
Used to specify a color in terms of red, green, blue, and alpha
HEX
Used to specify a color in terms of hexadecimal
HSL
Used to specify a color in terms of hue, saturation, and lightnessHSLA
Used to specify a color in terms of hue, saturation, lightness, and
Color Names
Used to specify a color by its name

Backgrounds

background-color
Used to specify the background color of an element
background-image
Used to specify the background image of an element
background-repeat
Used to specify whether the background image should be repeated or not
background-position
Used to specify the position of the background image
background-attachment
Used to specify whether the background image should be fixed or scroll with the rest of
background
Used to specify all the background properties in one property

Borders

border-width
Used to specify the width of the border
border-style
Used to specify the style of the border
border-color
Used to specify the color of the border
border
Used to specify all the border properties in one property
Border Radius
Used to specify the radius of the border

Margin

margin
Used to specify the margin of an element

Padding

padding
Used to specify the padding of an element

Box Model

  • Content
  • Padding
  • Border
  • Margin

Display

display like block, inline, flex, none e.t.c
Used to specify the display of an element
