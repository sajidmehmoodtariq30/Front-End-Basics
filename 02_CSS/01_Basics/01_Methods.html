Methods to add css in an html document.

There are 3 ways to add css in an html document.

Inline CSS

Inline css is used to add css to a specific element. It is written in the element's tag.

Internal CSS

Internal css is used to add css to a specific element. It is written in <style> tag.

External CSS

External css is used to add css to a specific element. It is written in a seperate css file included with a<link> tag. in html
